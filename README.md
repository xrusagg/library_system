Library Management System (Flask)

A simple web application built with Flask and PostgreSQL for managing a library system.
Users can add and view books, authors, and members, and perform custom queries such as books by author and available books.

Features:

-Flask framework
-PostgreSQL database (Render)
-Jinja2 templates
-Add and view Authors, Books, Members
-Custom queries (Books by Author, Available Books)

Run the App:
pip install flask flask_sqlalchemy psycopg2-binary python-dotenv
python app.py


Open in browser:
http://127.0.0.1:5000

Technologies:
Flask, PostgreSQL, SQLAlchemy, HTML, CSS
